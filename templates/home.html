I present to you my project "Django Library" on which I have been working for 3 months.

I was able to improve all the necessary skills and get closer to my goal of becoming an engineer.

I wanted to make an application that would be as user-friendly as possible, so during the development I was able to improve my skills in front-end development well, mastered CSS and Bootstrap.

This project contains 2 types of authorization: directly in the service and through social networks. All workarounds to the pages are closed, you get access to advanced functionality only after authorization. You can explore the list of authors and books that the Django Library provides. You can also order a book, specifying the date when you undertake to return the book. You will have a profile at your disposal, in which you can enter personal information and set an avatar. If you are a developer, you can click on the gear button in your profile to learn how to use the Django Library API.

A user can be in one of 4 groups: Guest, User, Librarian, Admin. Each group has its own set of rights, such as editing, deleting, adding.

During development, I encountered a number of problems, but in the course of solving them, I understood more and more the work of the technologies used, I saw my mistakes and shortcomings.

The following technologies were used in the project: Django, Django Rest Framework, PostgreSQL, HTML & CSS, Bootstrap, Docker, Docker-compose
