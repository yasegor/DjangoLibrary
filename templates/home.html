Ласкаво прошу у «Django Library». Це – проєкт, над котрим я працював протягом 3-х місяців.

За час розробки я зміг покращити важливі для мене навички та наблизитись до своєї цілі - стати інженером.

Моєю метою було створити застосунок, який би був максимально зручним для користувача, тому я навіть трішки поглибився у верстку, і засвоїв HTML, CSS та Bootstrap.

Цей проєкт містить 2 види авторизації: безпосередньо в сервісі та через соціальні мережі. В процесі реєстрації користувач отримає лист на пошту з посиланням-підтвержденням. Це один з методів безпеки від бот-атак. Також, при авторизації, користувач має пройти капчу.

Після реєстрації ви отримаєте власний профіль, в якому зможете вказати особисту інформацію і встановити аватар.

Якщо Ви розробник - натисніть на кнопку у вигляді шестерні для того, щоб дізнатися, як використовувати API Django Library.

Деякі функції обмежені правами доступу. В проекті реалізовані 4 групи прав: гість, юзер, бібліотекар, адмін. Саме тому дані знаходяться у повній безпеці

В проєкті було використано такі технології: Django, Django Rest Framework, PostgreSQL, HTML / CSS & Bootstrap, Docker, Docker-compose.
